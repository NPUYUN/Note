# GitHub操作
[GitHub官网](http://github.com/)

## 创建远程库
- 远程库名称一般与本地库一样
- 远程库基于用户名，一个用户下不能有两个重名的远程库

## 常见操作
### 创建别名
查看所有远程库别名
```bash
git remote -v
```

创建别名
``` bash
git remote add 别名 远程库地址
```

### 推送本地库到远程库
``` bash
git push 别名/远程库链接 分支
```

### 拉取远程库到本地库
``` bash
git pull 别名/远程库链接 分支
```
拉取动作自动提交本地库，即远程库和本地库代码同步
> 要时刻保持本地库永远为最新状态！！！

### 克隆远程库到本地
``` bash
git clone 远程库链接
```
克隆会自动：
- 拉取代码
- 初始化本地仓库
- 创建别名(默认为**origin**)

> Clone不需要登陆

### 邀请加入团队
#### 团队内协作
在设置中邀请成员后，该成员才能参与更改远程库代码

#### 跨团队协作
在须协作项目中点击 **fork** ，即可将项目放置在自己用户中
点击 **Pull request** 即可提交回去，在该界面可以支持聊天
提交合并申请即可合并修改后的代码

## SSH免密登录
生成SSH密钥目录
``` bash
ssh-keygen -t rsa -C Github账户.com
```
`.pub`为公钥
