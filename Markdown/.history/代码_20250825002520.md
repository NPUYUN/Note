# 代码的书写与注意事项
## 对于片段式的函数、变量、指令等
用两个反引号`` ` ``将该片段围起来
例如：
`printf()`

## 特殊字符转义
当需要在行内代码中显示反引号或者其他字符时，需要进行转义处理

- 使用 **双反引号** 包围单反引号
- HTML标签：\<div>元素
- 数学符号：计算x + y = z
- 特殊符号：使用&nbsp: 表示空格

## 代码区块
### 缩进式代码块
代码块使用 **四个空格** 或者一个 **制表符** 
语法格式：
文本段落

    缩进式代码块
    缩进式代码块
    缩进式代码块

文本段落


### 三反引号代码块
用```包含一段代码，可指定一种语言，并采用这种语言的常用语法高亮

> 注意：对于缩进式代码块以及未选择语言的三反引号代码块
> 1. 前后需要空行分隔
> 2. 所有代码行必须保持一致的缩进
> 3. 不支持语法高亮
> 4. 在列表中使用时需要8个空格缩进

- 常用语言标识符列表  
    - javascript / js ――JavaScript
    - python / py ――Python
    - html ――HTML
    - css ――CSS
    - sql ――SQL
    - json ――JSON
    - xml ――XML
    - yaml / yml ――YAML
    - bash / shell ――Shell脚本
    - java ――Java
    - cpp / c++ ――C++
    - csharp / c# ――C#
    - php ――PHP
    - ruby/ rb ――Ruby
    - go ――GO
    - rust ――Rust
    - typescript / ts ――TypeScript


- 代码块高级特性
    - 行号显示：在所选语言（无则直接）后加上`{.line-numbers}` 或者`showLineNumbers`
    - 代码差异对比：Diff语法（在前三反引号后加diff）显示代码的添加（+），删除（-）和修改，在所在行前加上对应符号即可

例如：
```cpp
#include<iostream>
using namespace std;
int main(){
    int n;
    cin >> n;
    cout >> n;
}
```

```cpp {.line-numbers}
 #include<iostream>

int main(){
    int n;
    std::cin >> n;
    std::cout << n;
}
```

```diff
#include<iostream>
+ using namespace std;

int main(){
    int n;
-   std::cin >> n;
+   cin >> n;
-   std::cout << n;
+   cout << n;
}
```
