# 标题的使用与注意事项

## 使用 = 和 - 标记一级和二级标题

- 一级标题：在文本下加 ============

- 二级标题：在文本下加 ---------------------

> 通过shift + 回车 实现文本下加

例如：

这是二级标题
---

## 使用#标记标题

- \# 一级标题

- \#\# 二级标题

- \#\#\# 三级标题

- 以此类推



## 3. 注意事项

- 符号和文字之间需加上**空格** 

- \#必须放在**行首** 

- 一个章节一级标题**唯一** 

- 标题锚点：[介绍部分\](对应标题加上\#\)
