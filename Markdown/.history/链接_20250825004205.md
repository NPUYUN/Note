# 链接的使用与注意事项
## 一般的链接使用
\[链接名称](链接地址)
\[链接文字](链接地址 "可选的标题")
\<链接地址>
> 可选的标题为鼠标悬停显示的

例如：
[README](http://www.runoob.com)
<http://www.runoob.com>
[README](http://www.runoob.com "你好")

## 参考链接
文本\[链接文字][参考标签]
\[参考标签]: URL "可选标题"
当链接文字与参考标签相同时，省略后者
优势：重复使用链接

例如：
README [Google][]

[Google]: http://www.google.com/

## 自动链接识别
直接输入网址或者邮箱，或者使用尖括号包含均可被markdown识别

例如：
www.runoob.com

## 锚点链接
\[介绍文本](章节（带#）)
通常作为目录放在开头
手动创建锚点：\<a id='锚点名称'>\</a>
页面顶部返回链接：\[回到顶部](#)
