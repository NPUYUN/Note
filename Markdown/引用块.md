# 引用块的使用与注意事项

## 单级引用的使用

\> 引用内容

只在第一行使用`>`, 其余行会自动包含在引用中

例如:

> README



## 多级嵌套引用

\> 最外层

\> \> 第一层嵌套

\> \> \> 第二层嵌套

以此类推

例如:

> README
> 
> > README
> > 
> > > README



## 引用块与其他

- 引用块可以与列表混用

- 引用快内几乎可以包含所有其他markdown元素

例如:

1. 与列表混用:
- README
  
  > README
  > 
  > README
  > 
  > - README



2. 包含标题:

> ## README
> 
>  
> 
> README



3. 包含代码:

> ```bash
> npm install
> npm start
> ```



4. 包含链接与图片

> &#x1f4da; **推荐阅读**
> 
> 详细信息请参考 [官方文档](https://example.com)
> 
> ![示例图片](./images/example.png)



5. 包含名言警句:

> "在软件开发中，最昂贵的错误就是构建正确的系统错误的方式，或者构建错误的系统正确的方式。"
> 
>  
> 
> ― Barry Boehm，软件工程专家

>

> 
> 连接数据库失败，请检查网络连接或联系系统管理员。
