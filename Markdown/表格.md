# 表格的书写与注意事项
## 基本语法
用`|`来分割不同的单元格，用`-`来分隔表头和其他行，如下：
\| 表头 \| 表头 \|
\| -------\| ------ \|
\| 单元格\| 单元格\|
\| 单元格\| 单元格\|
例如：

| 年级 | 绩点 | 
| ---- | ---- |
| 大一上| 3.7 |
| 大一下| 3.96 |

## 对齐方式
- `---:` ：设置内容和标题栏居右对齐
- `:---` ：设置内容和标题栏居左对齐
- `:---:` ：设置内容和标题栏居中对齐
如下：

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

## 复杂的表格处理
表格单元格内可以使用大部分Markdown语法
- 处理长文本：勤用`<br>`进行换行 以及 缩进和链接
- 特殊字符需要转义
- 表格美化：使用emoji和符号、对复杂的表格进行分解
