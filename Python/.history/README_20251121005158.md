# Python 学习与算法资源包

一个面向学习与实践的 Python 资源集合，涵盖常用科学计算库、数值分析、优化与运筹、仿真与随机过程、时间序列、机器学习、统计与回归、聚类与降维、贝叶斯方法等主题。所有脚本提供最小可运行示例，便于快速上手与查阅。

## 目录结构

- `常用库/`
  - `Numpy/`：数组创建、切片与索引、广播、线性代数、IO 等
  - `Pandas/`：`Series`/`DataFrame`、读写 CSV/Excel/JSON、清洗与聚合、可视化
  - `Matplotlib/`：折线/散点/柱状/直方/饼图、子图、中文显示、网格与标注
  - `Scipy/`：科学计算与优化等基础示例
- `常用算法/`：跨主题示例索引与专题说明
- `仿真与随机/`：蒙特卡洛、离散事件仿真、队列论、马尔可夫链、主体仿真、LHS
- `优化与运筹/`：线性/整数/非线性/凸优化、最短路、网络流、动态规划、多目标优化
- `启发式与元启发式/`：SA、TS、PSO、ACO、GA 等元启发式算法
- `数值计算/`：常/偏微分方程、有限差分/体积/元、数值积分、求根、BFGS、Euler、RK4、BVP
- `时间序列/`：ARIMA/SARIMA、GARCH/EGARCH、ETS、状态空间(Kalman)、Prophet
- `机器学习/`：KNN、决策树、随机森林、梯度提升、SVM、MLP、RNN、XGBoost、LightGBM、CatBoost
- `统计与回归/`：OLS、GLM、逻辑回归、稳健回归、岭/套索、分位数回归
- `聚类与降维/`：KMeans、DBSCAN、层次聚类、谱聚类、PCA、SVD、LDA、tSNE、UMAP
- `贝叶斯方法/`：MCMC、Metropolis-Hastings、Gibbs、变分推断、层次模型
- 根级文档：`README.md`、`初学笔记.md`、`初学笔记.pdf`

> 各子目录通常包含一个 `README.md` 和若干示例脚本，每个脚本以中文主题名并在括号中给出英文名，便于检索与对照。

## 环境要求

- Python `>=3.9`（建议使用虚拟环境）
- 科学计算与机器学习常用包（按需安装）：
  - `numpy`、`pandas`、`matplotlib`、`scipy`
  - `scikit-learn`、`statsmodels`
  - `xgboost`、`lightgbm`、`catboost`
  - `prophet`（时间序列）
  - `networkx`（图算法）
  - `umap-learn`（降维）

创建并激活虚拟环境（示例）：

```bash
python -m venv .venv
# Windows PowerShell
.\.venv\Scripts\Activate.ps1
# macOS/Linux
source .venv/bin/activate
```

按需安装依赖：

```bash
pip install numpy pandas matplotlib scipy scikit-learn statsmodels
pip install xgboost lightgbm catboost prophet networkx umap-learn
```

## 快速开始

以时间序列中的 `ARIMA.py` 为例：

```bash
python 时间序列/ARIMA.py
```

以优化中的 `线性规划(LP).py` 为例：

```bash
python 优化与运筹/线性规划(LP).py
```

以绘图中的 `散点图.py` 为例：

```bash
python 常用库/Matplotlib/初步教程/散点图.py
```

所有脚本均尽量自包含（必要依赖需先安装），运行后会在终端或图形窗口中展示结果。
