# Python 学习与算法资源包

本资源包聚合了常用 Python 科学计算与数据分析库的入门示例，以及覆盖优化、统计回归、时间序列、机器学习、仿真与数值计算等领域的算法模板，适合自学与快速查阅。

## 目录结构
- `常用库/`：Numpy、Pandas、Matplotlib、Scipy 的入门脚本与简要索引。
- `常用算法/`：各类算法的大纲与示例脚本（优化、时序、聚类、贝叶斯、图模型等）。
- `仿真与随机/`、`优化与运筹/`、`时间序列/`、`机器学习/`、`数值计算/` 等：按主题分类的可运行示例。
- `初学笔记.md`、`初学笔记.pdf`：学习笔记与导览。

## 环境依赖
- Python 3.9+（推荐 3.10/3.11）。
- 基础库：`numpy`、`pandas`、`matplotlib`、`scipy`、`scikit-learn`、`statsmodels`。
- 按需安装：
  - 梯度提升/集成：`xgboost`、`lightgbm`、`catboost`
  - 时间序列：`pmdarima`、`prophet`
  - 优化与图算法：`pulp`、`pyomo`、`cvxpy`、`ortools`、`networkx`
  - 仿真与采样：`simpy`、`SALib`
  - 贝叶斯：`pymc`、`cmdstanpy`
  - 可视化与降维：`umap-learn`

示例安装（按需选取）：
```
pip install numpy pandas matplotlib scipy scikit-learn statsmodels
pip install xgboost lightgbm catboost pmdarima prophet
pip install pulp pyomo cvxpy ortools networkx simpy SALib umap-learn
pip install pymc cmdstanpy
```

## 快速开始
- 运行库示例：
  - `python 常用库\Numpy\初步教程\创建数组.py`
  - `python 常用库\Matplotlib\初步教程\Pyplot.py`
- 运行算法模板：
  - `python 时间序列\ARIMA.py`
  - `python 优化与运筹\线性规划(LP).py`
  - `python 机器学习\决策树(DecisionTree).py`

## 常见用途
- 快速回顾 API 与典型用法，复制示例直接改造。
- 按问题类型选型（参考 `常用算法/README.md` 中的大纲与流程建议）。
- 将脚本作为最小可运行原型，替换数据与参数后用于实验。

## 注意事项
- 文件与文件夹包含中文命名，确保终端/IDE 使用 UTF-8 编码。
- `prophet`、`cvxpy` 等库在 Windows 上可能安装较慢或需要额外依赖，建议在虚拟环境中按需安装。
- 示例脚本以教学为主，未做生产级异常处理与日志，请按项目需求完善。
